R-Share

This is by far my most favorite project yet. It is a ride sharing start-up that I started with a few friends. The whole idea is to connect students going back home from college with another college student going to that same city. After being connected, they just split the gas bill and go their respective ways. We created this in a React stack with Firebase for the database system. I learned a lot about styling and the lifestyle of a product through this project.

Deeper Dive:
  • Tech Stack: React.js, Firebase (Firestore), Python, Flask, JavaScript, HTML, CSS, SQLite
  • Frontend: Built with React.js for a dynamic, responsive user experience. Utilized React Router for navigation and Material UI for component styling.
  • Backend: Flask RESTful API for ride management, user authentication, and profile management. Integrated with Firebase for real-time database and notifications.
  • Database: Used Firebase Firestore for scalable, cloud-based data storage. SQLite for local development and prototyping.
  • Authentication: Implemented secure login and registration using Firebase Auth and JWT tokens for API access.
  • Features: Ride creation, search, and matching algorithm to connect students. Gas bill splitting and payment tracking. Push notifications for ride updates.
  • Design: Mobile-first, responsive web design for seamless experience on all devices.

Notionary

This is another very exciting project that I have worked on (click the image to go to our trailer). We started this app off at a hackathon, but liked it so much that we ended up finishing the project. We used Next.js, Typescript, and Tailwind CSS with Groq LLM to generate the quizzes. The reason why I like this project so much is because we were able to connect it directly to our own notes and get quizzes from those notes. The biggest thing I learned through this project is Docker.

Deeper Dive:
  • Tech Stack: Next.js, TypeScript, Tailwind CSS, Groq LLM, Node.js, Notion API, Docker, HTML, CSS
  • Frontend: Next.js and Tailwind CSS for fast, modern UI. TypeScript for type safety and maintainability.
  • Backend: Node.js server integrates with Notion API to fetch notes and generate quizzes. Groq LLM used for AI-powered quiz generation.
  • Features: Automated quiz creation from Notion databases, customizable question formats, export to PDF/CSV, and direct integration with productivity tools.
  • AI: Leveraged Groq LLM for natural language processing and quiz generation from raw notes.
  • Design: Responsive, mobile-friendly interface. Customizable themes and layouts.
